Things to keep in mind! Important!

navbar class should be always included to the navbar tag.
navbar-light, changes the font color to light.
bg-light, changes the navbar background color to light.
The brand should be in a anchor tag, with a class of navbar-brand.

Navbar with links

3 components, navbar-brand, navbar-toggler(Button), navbar-collapse.

The navbar-toggler(button) should have class="navbar-toggler" type="button" data-toggle="collapse" data-target="#idOfTheNavbarCollapse".

Inside the navbar-collapse, there will be the links that is wrapped in a div that has a class of navbar-nav. The links should have a class of nav-link.
